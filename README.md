# HealthTrack React Project

## Description
A simple react web app to track your daily health activities and calculate your BMI.
Users can view features, contact, and use the BMI calculator.

## Setup Instructions
1. Clone the repo:
   git clone https://github.com/12231665-nancy/healthtrack-react-project
2. Go to the project directory:
   cd healthtrack-react-project
3. Install dependencies:
   npm install
4. Start the app:
   npm start
